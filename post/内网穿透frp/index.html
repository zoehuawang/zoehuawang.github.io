通过访问服务端代理到客户端上，详见 https://github.com/fatedier/frp

  • 服务端示例，版本 v0.62.0：

配置文件 frps.toml

bindAddr = "172.23.33.3"
bindPort = 7788
auth.method = "token"
auth.token = "wellcloudavqoub89"

启动服务端

./frps -c frps.toml
  • 客户端示例，版本 v0.62.0：

配置文件 frpc.toml

serverAddr = "172.23.33.3"
serverPort = 7788
auth.method = "token"
auth.token = "wellcloudavqoub89"

[[proxies]]
name = "test-tcp"
type = "tcp"
localIP = "172.23.33.11"
localPort = 22
remotePort = 6622

{{- range $_, $v := parseNumberRangePair "18627-18629" "18627-18629" }}
[[proxies]]
name = "udp-{{ $v.First }}"
type = "udp"
localIP = "172.23.33.11"
localPort = {{ $v.First }}
remotePort = {{ $v.Second }}
{{- end }}

启动客户端

./frpc -c frpc.toml
